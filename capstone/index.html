Capstone Project

Capstone is a research-based project that students in my school complete during their senior year, and my project is/is going to be computer science-based (as expected). Specifically, I will be learning about and create a project with machine learning. The goal of machine learning is to create algorithms that operate on data to create a model that represents the data, which can be used to make decisions and predictions without the need of static program instructions written by programmers for a specific task. Within machine learning, I plan on focusing on deep learning, which is an active area of research in machine learning. Deep learning is concerned with the learning of useful representations of data using models with multiple layers.
