Swing

In November 2014 I learned how to use Swing, which is the Java API for creating GUIs (Graphical User Interfaces). The purpose of GUIs is very straightforward - they present information in a way that is more useful and accessible for users of a program. Most people do not want to deal with text input and output through a console, so GUIs are created for them. GUIs also improve the appearance of a program, so learning how to create them was important in my ability to create professional-looking programs. There are a variety of different layouts and components within Swing that I learned how to use, all of them with applications to different GUIs for different programs.
