Data Structures

The entire purpose of programming can be described as dealing with data. Organizing, displaying, processing, analyzing, and more can be done with data, but data is always at the core of programs. Therefore, learning how to implement and apply data structures in Java was a very beneficial experience. To do this, I followed the curriculum for AP Computer Science AB (which is no longer offered) from November 2014 to February 2015.

Implementation

One of the focuses of the AB curriculum was to learn how to implement data structures to gain a greater understanding of how they work, and consequently, how they should be applied. For each of the data structures I learned about, I learned how to implement them from scratch.

Application

Learning how to implement data structures was an important part of the Comp Sci AB curriculum, but that knowledge would be useless if I didn't also learn how to apply data structures to specific problems. Throughout the curriculum, I learned how to apply the data structures I was learning about to problems. To give one example, I learned how to use stacks to evaluate mathematical expressions written in postfix form.
