File Input/Output

During October 2014, I learned how to do input and output with files through Java programs. Since input and output are essential to the operation of programs, learning how to do file IO in Java was important in the development of my ability to write programs that are useful. File IO is a crucial part of most large programs, as input and output should not always come from and go to program users.

Text

The first thing I learned in this process was how to do input and output with text files. Writing to and reading from text files can be useful when the content of the files needs to be in a user-readable format, even when the format needs to be something such as HTML, as in the program shown below.

Binary

While text files can be useful for some applications, the majority of files that are used by computers are binary files that are not user-readable. Try opening an image file in a text editor - most of the text will make no sense. Instead, programs are used that read the binary from the file and present it in a format that is useful to humans. This is how most files are, so learning how to work with binary files in Java was very important.
