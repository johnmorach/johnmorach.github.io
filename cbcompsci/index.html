CB Computer Science

CB Computer Science Principles was a class that I took during my freshman year of high school. In this class I learned many of the principles of computer science that are essential to learning more advanced topics in CS. This class was a pilot for the College Board, which is developing the Computer Science Principles course with the intent of making it an Advanced Placement course. Some of the work that I did in this class is available online, including some of the apps I created using Android AppInventor and the work I did in JavaScript using Code in the Browser.
